Данная программа написана Данилом Суровым. Эта программа реализует автоматический алгоритм метода дифференциальных рент для решения транспортных задач.

Стек технологий: Pyton 3.11, JS, HTML, CSS

Для реализации графического интерфейса использовался язык HTML, чтобы связать front-end с back-end, используется открытая библиотека EEL для Python.

Во вкладке домашняя страница находится изменяемая таблица, в которую пользователь может вводить свои данные для задачи, после чего может вычислить результат.

Во вкладке решение находится пошаговое решение в таблицах и вычисление финальной суммы.

Во вкладе о программе находится базовая информация о программе, стеке и создателе
